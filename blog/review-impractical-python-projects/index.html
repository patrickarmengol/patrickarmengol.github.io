Review: Impractical Python Projects

2022-11-18

This is my short review of “Impractical Python Projects” by Lee Vaughan.

About

The book is meant to fill the post-beginner gap with playful projects. It’s available at https://nostarch.com/impracticalpythonprojects.

Likes

  • the projects were fun
  • science and math focus, but not too mathy
  • fits target audience
  • humorous writing style
  • unique projects that are usually not found in recommendations to beginners

Dislikes

  • code sometimes seemed a bit too convoluted
  • many mistakes in code not listed in latest errata

Conclusion

It was good. It inspired me to do some further experimentation, especially into NLP.
