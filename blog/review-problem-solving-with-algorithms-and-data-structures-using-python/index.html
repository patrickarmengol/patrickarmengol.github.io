Review: Problem Solving with Algorithms and Data Structures using Python

2022-09-30

This is my short review of “Problem Solving with Algorithms and Data Structures using Python” by Brad Miller and David Ranum.

About

This book is published online for free as an interactive textbook at https://runestone.academy/ns/books/published/pythonds3/index.html

I found this book through a recommendation on reddit when I was searching for a somewhat comprehensive DSA book with Python code.

Things I liked

  • fairly comprehensive
  • code your own data structures
  • tons of suggested exercises at the end of each chapter
  • real world relevance discussed

Things I disliked

  • only a few mentions of existing libraries that implement/optimize certain algorithms or data structures
  • coding style was a bit strange; some implementations were a bit convoluted and focused too much on the example problem task
  • many errata, especially in the figures and code
  • some explanations were lacking

Conclusion

This book ended up being very close to what I was looking for. I was able to review and refresh my memory of various data structures and algorithms that I learned in uni via C and Java.

I often found myself straying from the book and looking elsewhere for better explanations, pseudocode, more generic Python implementations, and animated visualizations.

I’d like to keep a repo with simple versions of many of the discussed DSAs for reference, kind of like this and this. I should also probably take a look at how the Python standard library already does common data structures.

Eventually, I think a good project for interview prep would be to visualize DSAs with something like manim or p5.
