This is my short review of "Automate the Boring Stuff with Python" by Al Sweigart. 

## About

This book is aimed at absolute beginners of python and/or programming. I read it as the first book in my python re-learning journey. Prior to reading, I already knew quite a bit of python, but I chose to read it page by page, following all the examples, and doing most of the projects, because I suck at python and want to suck a little bit less. I am nowhere near an expert level so any criticism I give can be taken as "a beginner's perspective". 

## Things I liked

- great explanations of beginner python and programming concepts
- shows off practical applications of what's being taught
- loads of projects; some guided
- free online version (though I did "buy" this book in a humblebundle)
- Zophie

## Things I disliked

- camelCase
- a lot of the modules introduced were written by the author; likely stuff I will never use in the future
- very focused on automating tasks
- author uses Windows (though consideration is given to non-Windows readers)

## Conclusion

Overall, I think it was pretty good, with some annoyances. If someone is looking for a free resource, I'll point them here; else, I think "Python Crash Course" by Eric Matthes gives a more comprehensive introduction.
