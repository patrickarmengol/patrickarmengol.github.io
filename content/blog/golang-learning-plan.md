## Why

Golang (yes I'm going to call it golang because it makes it easier to search and because I associate the word _go_ with the board game) is a nice language. It seems simple but capable. I want to use it to dive deeper into the world of systems programming. Up until now I have been using python for basically anything and everything and I'm quite comfortable with it, but it often doesn't really feel like the best tool for the job. If I want to get serious about backend web dev, golang is a no-brainer.

## Goals

I want to be really proficient at golang. I want golang to be my default for backend web dev work.
